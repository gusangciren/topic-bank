# 存入选题库

> 把他人的好文章存进你的选题库；把自己的思考碎片追加到已有文章中

---

## ⚙️ 安装后请编辑 `skill.json`

fork 本项目后，只需编辑 `skill.json` 即可定制：

| 字段 | 说明 | 示例 |
|------|------|------|
| `trigger_words` | 触发词列表（说任一词激活） | `["存入选题库", "存入素材库", "追加到"]` |
| `target_dir` | 存放文件夹（自动创建） | `"~/Desktop/定时发布/"` |
| `filename_date_prefix` | 文件名是否以日期开头 | `true` |

---

## 使用方式

### 新建文章

```
存入选题库

标题：文章标题
正文：这里是文章正文内容...
```

正文是必填项。

### 追加到已有文章

```
追加到写作就是思考

这里是你想追加的新内容...
```

触发词 `追加到` 后面跟文件名关键词（模糊匹配，无需完整文件名），我会找到对应的 `.md` 文件，在文末追加你的内容并更新时间戳。

### 存入后可以打标签

```
存入选题库

标题：文章标题
正文：文章正文内容...
标签：#创业 #写作 #思维模型
```

标签会附在文末。

## 安装方式

```bash
git clone https://github.com/gusangciren/topic-bank.git \
  ~/.qclaw/skills/topic-bank
```

安装后编辑 `skill.json` 即可使用。

## 安全说明

- 仅做文件写入，不访问网络
- 文件名自动处理特殊字符，重名自动加序号
- 零外部依赖，Python 3.6+ 内置库即可
